This is a collection of my current self-hosted stack running on my home server. It primarily revolves around my media server setup, which includes Jellyfin, Radarr, and Sonarr. These services work together to manage, organize, and deliver media content efficiently.

The system is designed to run reliably with minimal maintenance, giving me full control over my library and how it's accessed. Everything is containerized for better isolation and easier updates, and the stack continues to evolve as I refine the setup.
